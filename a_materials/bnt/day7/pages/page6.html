üé® HTML <canvas> Tag - Explained for Students

The <canvas> tag in HTML is used to draw shapes, text, images, and more using JavaScript. It is very useful for creating graphics like charts, games, and animations.

üîß Basic Syntax

<canvas id="myCanvas" width="400" height="200"></canvas>

<script>
  const canvas = document.getElementById("myCanvas");
  const ctx = canvas.getContext("2d");
  // Drawing code goes here
</script>

‚úçÔ∏è Drawing a Simple Line

// JavaScript
const canvas = document.getElementById("lineCanvas");
const ctx = canvas.getContext("2d");

ctx.moveTo(10, 10);       // Starting point
ctx.lineTo(290, 140);     // Ending point
ctx.stroke();             // Draw the line

üü• Drawing a Filled Rectangle

// JavaScript
const canvas = document.getElementById("rectCanvas");
const ctx = canvas.getContext("2d");

ctx.fillStyle = "blue";         // Fill color
ctx.fillRect(50, 40, 120, 80);  // x, y, width, height

üîµ Drawing a Circle

// JavaScript
const canvas = document.getElementById("circleCanvas");
const ctx = canvas.getContext("2d");

ctx.beginPath();
ctx.arc(150, 75, 50, 0, 2 * Math.PI); // x, y, radius, start angle, end angle
ctx.fillStyle = "orange";
ctx.fill();
ctx.stroke();

üñãÔ∏è Writing Text on Canvas

// JavaScript
const canvas = document.getElementById("textCanvas");
const ctx = canvas.getContext("2d");

ctx.font = "24px Arial";
ctx.fillStyle = "green";
ctx.fillText("Hello, Canvas!", 50, 50);

üìö Common Canvas Methods

  • moveTo(x, y) ‚Äì Moves the path to start from (x, y)
  • lineTo(x, y) ‚Äì Draws a line to (x, y)
  • arc(x, y, radius, startAngle, endAngle) ‚Äì Draws an arc/circle
  • fillRect(x, y, width, height) ‚Äì Draws filled rectangle
  • fillText(text, x, y) ‚Äì Writes text
  • stroke() ‚Äì Outlines the current path
  • fill() ‚Äì Fills the shape with color

üí° Tips for Students

  • Always use getContext("2d") to draw in 2D.
  • JavaScript is required to control the canvas.
  • The canvas won't show anything unless you draw on it with JS.
  • Try combining shapes to make patterns or designs!
