Difference Between display: none and visibility: hidden

1. display: none;

This property removes the element from the document layout. It will not occupy any space on the page.

2. visibility: hidden;

This property hides the element, but the space it occupies will still be visible (blank space remains).

Live Example:

Box with display: none

I am hidden (display: none)
<div class="box none-box">...</div>
.none-box { display: none; }

Box with visibility: hidden

I am invisible (visibility: hidden)
<div class="box hidden-box">...</div>
.hidden-box { visibility: hidden; }

Summary Table:

Property Element Hidden? Space Occupied?
display: none; Yes No
visibility: hidden; Yes Yes

Use Cases:

  • Use display: none when you want to completely remove the element from layout (e.g., hide a menu item).
  • Use visibility: hidden when you want to maintain the layout but hide the content (e.g., hiding something temporarily).
