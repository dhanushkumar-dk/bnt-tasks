Media Queries in HTML/CSS

üß† What Are Media Queries?

Media queries are used in CSS to apply different styles based on the device's screen size, resolution, orientation, or other features. This enables responsive design that adapts to phones, tablets, laptops, and desktops.

üßæ Basic Syntax

@media media-type and (condition) {
  /* CSS styles here */

  CSS_Selector {
    attribute:value;
  }
}

Example:

@media screen and (max-width: 1200px) {
  .box {
    background-color: #27ae60; # green color
  }
}

Example:

@media screen and (max-width: 992px) {
  .box {
    background-color: #f39c12; # orange color
  }
}

This applies a light gray background only when screen width is 768px or less.

üìè Common Breakpoints

  • max-width: 1200px - Large screens
  • max-width: 992px - Tablets
  • max-width: 768px - Large phones
  • max-width: 576px - Small phones
  • max-width: 400px - Extra small screens
