Understanding HTML5

A Simple Guide for Students

What is HTML5?

HTML5 is the latest version of HyperText Markup Language, the standard language for creating web pages. It brings new elements, APIs, and capabilities to build modern, responsive, and interactive websites.

New Features in HTML5

  • Semantic Tags: <header>, <footer>, <section>, <article>, etc., for better structure.
  • Multimedia: Direct support for <audio> and <video> tags.
  • Form Enhancements: New input types like email, date, range.
  • Canvas: For drawing graphics via JavaScript.
  • Geolocation API: Access user’s location.
  • Local Storage: Store data on browser without cookies.

Semantic Elements

HTML5 introduced semantic tags to better describe the content:

  • <header> – Introductory content
  • <nav> – Navigation links
  • <section> – Thematic grouping of content
  • <article> – Independent, self-contained content
  • <footer> – Footer of the page or section

HTML5 Audio and Video

HTML5 allows embedding multimedia without plugins:

Example: Audio

<audio controls>
  <source src="song.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

Example: Video

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

HTML5 Canvas API

The <canvas> tag is used to draw graphics using JavaScript. 

<canvas id="myCanvas" width="200" height="100"></canvas>

<script>
  const canvas = document.getElementById("myCanvas");
  const ctx = canvas.getContext("2d");
  ctx.fillStyle = "blue";
  ctx.fillRect(20, 20, 150, 50);
</script>

HTML5 APIs (Examples)

  • Geolocation API: Access user’s current location.
  • LocalStorage API: Store data locally in key-value format.

Example: Geolocation

navigator.geolocation.getCurrentPosition(function(position) {
  alert("Latitude: " + position.coords.latitude);
});

Example: LocalStorage

localStorage.setItem("username", "student");
let user = localStorage.getItem("username");

HTML5 Form Input Types

HTML5 introduced new input types that improve user experience and validation.

  • <input type="email"> – Validates email format
  • <input type="date"> – Date picker
  • <input type="range"> – Slider control

Advantages of HTML5

  • No need for Flash or external plugins
  • Better accessibility with semantic tags
  • Faster page load and cleaner code
  • Mobile-friendly and responsive

Conclusion

HTML5 is powerful, flexible, and essential for modern web development. Mastering it helps you build structured, accessible, and interactive websites easily.
