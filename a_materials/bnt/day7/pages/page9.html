Understanding HTML Entities

HTML entities are used to display reserved characters in HTML. These characters either have special meaning in HTML (like < or >) or are invisible characters like spaces or symbols that are not found on a keyboard.

Why use HTML Entities?

  • To display reserved HTML characters (like <, >, &).
  • To insert special symbols (like ©, ®, €, ₹).
  • To insert invisible characters like non-breaking space.

Common HTML Entities

Less than <
Code: &lt;
Output: <
Greater than >
Code: &gt;
Output: >
Ampersand &
Code: &amp;
Output: &
Double Quote "
Code: &quot;
Output: "
Non-Breaking Space
Code: &nbsp;
Output: This  is  spaced → This  is  spaced

Currency Symbols

Rupee Symbol ₹
Code: &#8377;
Output: ₹
Euro Symbol €
Code: &euro;
Output: €
Pound Symbol £
Code: &pound;
Output: £
Copyright Symbol ©
Code: &copy;
Output: ©
Trademark Symbol ™
Code: &trade;
Output: ™

Bonus: Using Numeric Codes

Every entity also has a numeric version. For example:

Heart Symbol ♥
Code: &#9829;
Output: ♥

Summary

  • Entities begin with & and end with ;.
  • They are used to write characters that are otherwise interpreted by the browser.
  • They improve readability and ensure HTML behaves correctly.
